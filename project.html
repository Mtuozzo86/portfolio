Chat app

What I consider to be the most challenging app I have built on my own, with the help of numerous blogs from developers who learned how to piece information together to achieve goals have helped me to create my own version of a live messaging app that allows real time messaging. Front end created using React and back end Ruby on Rails to handle storing users and user's messages and list of chat rooms and groups. Ruby on Rails offers Action Cable. With Action Cable, I am able to use my learned knowledge of Ruby to implement WebSockets, and allow constant communication between the browser and the server.

With heroku soon to be doing away with the free tier of deployment, this app will soon be demonstrated in its development phase.

ThoughtSpace

One of my earlier projects to demonstrate authorization and authentication. Logging into any website or app are best handled knowing passwords and information are protected. This app simply allows users to create, edit, update and delete messages. Each user is only allowed to perform these actions if they are logged in, and can only perform these actions to their own messages. One user cannot edit or delete another user's message. Password protection is used by using features of rails gems, especially bcrypt.

This project helped with understanding user experience and the importance of security.
